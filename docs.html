API Documentation

Complete guide for using the Predictive Maintenance API

Quick Start

The Predictive Maintenance API provides machine learning-powered predictions for equipment failure analysis.

Base URL: {{ request.url_root }}

Content-Type: application/json

API Endpoints

POST /predict

Main prediction endpoint for machine failure analysis.

Request Body:
{
  "Temperature_C": 75.5,
  "Vibration_mms": 4.2,
  "Power_Consumption_kW": 150.3,
  "Operational_Hours": 8760,
  "Error_Codes_Last_30_Days": 5,
  "Oil_Level_pct": 85.5,
  "Coolant_Level_pct": 78.2,
  "Maintenance_History_Count": 12,
  "Failure_History_Count": 3,
  "Machine_Type": "Type_A"
}
Response:
{
  "Failure_Within_7_Days": false,
  "Failure_Probability": 0.2345,
  "Remaining_Useful_Life_days": 45.67
}

GET /api_status

Check API health and model loading status.

Response:
{
  "status": "running",
  "models_loaded": true,
  "message": "🔧 Predictive Maintenance API is Running!"
}

Input Parameters

Parameter Type Description Example
Temperature_C float Operating temperature in Celsius 75.5
Vibration_mms float Vibration level in mm/s 4.2
Power_Consumption_kW float Power consumption in kilowatts 150.3
Operational_Hours float Total operational hours 8760
Error_Codes_Last_30_Days int Number of error codes in last 30 days 5
Oil_Level_pct float Oil level percentage (0-100) 85.5
Coolant_Level_pct float Coolant level percentage (0-100) 78.2
Maintenance_History_Count int Number of maintenance activities 12
Failure_History_Count int Number of previous failures 3
Machine_Type string Machine type (3D_Printer, AGV, CNC_Lathe, etc.) "3D_Printer"
AI_Override_Events int Number of AI override events 0
Installation_Year int Year of machine installation 2020
Last_Maintenance_Days_Ago int Days since last maintenance 30
Sound_dB float Sound level in decibels 65.5

Response Format

Field Type Description
Failure_Within_7_Days boolean Whether machine is predicted to fail within 7 days
Failure_Probability float Probability of failure (0.0 to 1.0)
Remaining_Useful_Life_days float Estimated remaining useful life in days

Code Examples

Single Prediction

Python - Single
import requests
import json

url = "{{ request.url_root }}predict"
data = {
    "Temperature_C": 75.5,
    "Vibration_mms": 4.2,
    "Power_Consumption_kW": 150.3,
    "Operational_Hours": 8760,
    "Error_Codes_Last_30_Days": 5,
    "Oil_Level_pct": 85.5,
    "Coolant_Level_pct": 78.2,
    "Maintenance_History_Count": 12,
    "Failure_History_Count": 3,
    "Machine_Type": "3D_Printer",
    "AI_Override_Events": 0,
    "Installation_Year": 2020,
    "Last_Maintenance_Days_Ago": 30,
    "Sound_dB": 65.5
}

response = requests.post(url, json=data)
result = response.json()
print(f"Failure in 7 days: {result['Failure_Within_7_Days']}")
print(f"Failure probability: {result['Failure_Probability']:.2%}")
print(f"Remaining life: {result['Remaining_Useful_Life_days']} days")

JavaScript - Single
const data = {
    "Temperature_C": 75.5,
    "Vibration_mms": 4.2,
    "Power_Consumption_kW": 150.3,
    "Operational_Hours": 8760,
    "Error_Codes_Last_30_Days": 5,
    "Oil_Level_pct": 85.5,
    "Coolant_Level_pct": 78.2,
    "Maintenance_History_Count": 12,
    "Failure_History_Count": 3,
    "Machine_Type": "3D_Printer",
    "AI_Override_Events": 0,
    "Installation_Year": 2020,
    "Last_Maintenance_Days_Ago": 30,
    "Sound_dB": 65.5
};

fetch('{{ request.url_root }}predict', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(result => {
    console.log('Failure in 7 days:', result.Failure_Within_7_Days);
    console.log('Failure probability:', result.Failure_Probability);
    console.log('Remaining life:', result.Remaining_Useful_Life_days, 'days');
});

import json

    {

        "AI_Override_Events": 0,

        "Last_Maintenance_Days_Ago": 30,
        "Sound_dB": 65.5
    },
    {

        "Power_Consumption_kW": 175.2,

        "AI_Override_Events": 2,
        "Installation_Year": 2018,
        "Last_Maintenance_Days_Ago": 45,
        "Sound_dB": 78.2
    }
]

response = requests.post(url, json=batch_data)
result = response.json()

print(f"Total predictions: {result['total_predictions']}")
print(f"Successful: {result['successful_predictions']}")

for prediction in result['batch_results']:
    idx = prediction['batch_index']
    print(f"\nMachine {idx}:")
    if 'error' not in prediction:
        print(f"  Failure in 7 days: {prediction['Failure_Within_7_Days']}")
        print(f"  Failure probability: {prediction['Failure_Probability']:.2%}")
        print(f"  Remaining life: {prediction['Remaining_Useful_Life_days']} days")
    else:
        print(f"  Error: {prediction['error']}")

Error Handling

The API returns appropriate HTTP status codes and error messages:

  • 200 OK: Successful prediction
  • 400 Bad Request: Invalid input data
  • 500 Internal Server Error: Server or model error
Error Response Format:
{
  "error": "Detailed error message"
}
